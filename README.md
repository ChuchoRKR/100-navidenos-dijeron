# 100 Navideños Dijeron
Proyecto Flutter generado por ChatGPT. App navideña tipo "100 Mexicanos Dijeron" con:
- 30 preguntas navideñas
- Modo equipos (Rojo vs Verde)
- Temporizador por pregunta
- Animación de nieve
- Soporte para sonidos (usa assets/sounds/*.mp3)
- Añadir preguntas personalizadas (guardadas en SharedPreferences)

## Cómo usar
1. Descarga y descomprime este proyecto.
2. Abre la carpeta en Android Studio (Flutter SDK debe estar instalado).
3. Ejecuta `flutter pub get`.
4. Agrega archivos de audio en `assets/sounds/` (opcional):
   - success.mp3
   - background.mp3
   - tick.mp3
5. Ejecuta la app en un emulador o dispositivo.

## Subir a GitHub
1. Crea un nuevo repositorio en GitHub.
2. Sube todo el contenido de esta carpeta.
3. Opcional: compila un APK y súbelo a Releases para compartir.
